This is a TCP server/client bundle that acts as a ATM.

Run server.py before client.py. 

There are four single-word commands associated with this program

deposit which allows users to put money into the bank. It will then ask the user how much they want to deposit
withdraw which allows users to take money out of the bank. It will then ask the user how much they want to withdraw
balance which allows users to check the balance in the bank
logout which logs the user out of the client 

The program will automatically make all the lowercase and remove any whitespace.
